# SSO-system

基于Web的单点登录系统，含Java的Web编程（JSP，Servlet，Filter，Listener）、设计模式、Tomcat服务器的使用

## 基本功能介绍

* **2个需要用户登录的应用系统(Web1和Web2)**:  登录后，主页面显示该系统已经登录的所有用户的信息和登录时间。并提供两个系统相互跳转的链接。一个应用系统可以登录多个用户

* **单点登录系统WebSSO**:  登录后，在主页面显示已经登录的所有用户的信息和登录时间。并提供跳转到web1和web2的链接，能够直接登录到web1或者web2

* 未登录状态下访问**Web1**或者**Web2**，跳转到单点登录系统**WebSSO**进行登录验证。登录成功后，跳转回**Web1**或**Web2**

* **Web1**、**Web2**和**WebSSO**可登出和注销。登出任何一个系统后，再次访问任何一个系统都需要重新登录。

* 应用系统具备账号注册、登录、登出、禁用，查看登录历史记录等管理功能。

## 设计模式

工厂模式、命令模式、拦截过滤器模式、传输对象模式、数据访问对象模式
